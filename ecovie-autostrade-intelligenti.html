ECOVIE, AUTOSTRADE INTELLIGENTI

Branding, illustrations / PoliMi / 2019

Scroll to see the photos and read full text
The work consists of the branding project for a smart highway management company (automated structures, technologies able to recycle clean energy...).
The project required the definition of the brand identity — values, mission, vision, contents — and its translation into a visual system that could be able to articulate itself on the different communication layers, B2B and B2C.

Specifically, B2C communication focuses on the sustainable travelling value and report it into a visual language that results poetic and dreamy thanks to the use of the illustrative technique.
